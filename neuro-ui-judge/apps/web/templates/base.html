Scientific framing. NeuroUI Judge does not measure preferences. TRIBE-derived features are population-level fMRI proxies for attention, hierarchy, load, and weak affect. Accessibility is enforced via WCAG-grounded deterministic checks, never neural inference. Human testing is required before any product decision.

This MVP uses a mock neural proxy by default. See tribe_adapter.py for how to plug in real TRIBE v2 inference.
